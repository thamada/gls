# nvidia-smi

an nvidia-smi simulator


## Install

```
pip install git+https://github.com/thamada/nvidia-smi.git
```

then, you can use the following commands:

```
nvidia-smi
nvidia-smi -a
deviceQuery
rocm-smi


nvidia-smi.h100x8
nvidia-smi.rtx2000ada
deviceQuery.rtx2000ada
deviceQuery.mi300x
```


## Uninstall

```
pip uninstall nvidia-smi-simulator
```

Enjoy!
